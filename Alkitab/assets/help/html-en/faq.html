How do I get the NIV, ESV, NKJV, or other translations of the Bible?

We did not include these versions because they are not free (as in liberty) or free (as in gratis), so we can not include it in this application. You can read it in, for example, www.biblegateway.com for those versions if you want to consult them.

How do I search for a word like "word" without getting results containing other terms such as "sword" or "words"?

In order to find an exact word without affixes or other words that are not expected, add a plus sign (+) immediately before the word. For example: +word.

Is this app also available for iOS or Windows or any other platforms?

No, this app is only available for Android. However, the source code is available at https://github.com/yukuku/androidbible if you want to port it to other platforms.
