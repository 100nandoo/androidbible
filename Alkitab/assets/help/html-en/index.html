When you run this application, you will immediately see the Bible text. Press the button below that is showing the verse address, like Genesis 1, to go to other parts of the Bible. Press the left and right button to move to the previous and next chapters.

Note: Some of the verses, especially in the book of Psalms, has numberings that differ between different translations and other translations (e.g. KJV). The chapter and verse numbers of the specific translation are written in parentheses.

Moving to Other Parts of the Bible

To move to another part of the Bible, you can press the verse address button, like Genesis 1

If No Navigation Buttons option is enabled, you can press the gray verse address bar above the text instead of the verse address button.

Hold (long-press) the verse address button to open the list of recent verses.

On the Direct tab, you can directly type the verse address. How to write the verse address is: Write the first 3 or so characters of the book name, then the chapter number, then a space or a colon, then the verse number. For example Luk 3:11 or 1Pe 2:5 (1 Peter 2:5).

However, for your convenience, you can also write it in the following ways:

  • Type just the chapter number to go to the chapter on the same book. Example: 8
  • Type the chapter number followed by the verse number. Example: 3 16, 12:4
  • Type the book name just enough. Example: songs (Song of Solomon), 1k (1 Kings), ps (Psalms)
  • Concatenate the book name and the chapter number: Joh3, Gen48

Bookmarks

Tap a verse and tap on the button that appears to open the a menu. On the menu there are options to Add bookmark. With bookmarks, you can mark verses that you think are important. You can also give a title and labels to each bookmark. Labels can be made by choosing Create label. Press MENU and select Bookmarks to see the list of bookmarks. Press one of the bookmarks to open it, or hold to change the title, labels, or to delete it.

By pressing MENU, you can also Backup or Restore. Use this to ensure that your bookmaks, notes, and highlights are stored properly in SD Card or USB Storage as a backup in case there is a problem with your device. Use Transfer to create a backup and send it to another application. This could be useful to move application data to other devices.

Search

You can find verses that contain certain words. Press the search button (with the magnifying glass symbol).

When the search screen opens, type the words that want to look for and click the button. The verses that contain all the words you entered will be displayed. For example, if you type god son, then even Genesis 4:25 would be displayed because it contains verses the word "god" and "son", although not contiguous. If you type just, you will also find "justice", "justify", etc.

  • Use quotes to search for exact phrases. For example, "our father".
  • Use plus sign (+) exactly before a word to search a whole word. For example, +eve will find the exact word "eve" or "Eve" but not "eventually" or "evening".

Devotion

This feature is only available in the Indonesian version of Bible for Android (search for "Alkitab" in Android Market).

Press MENU and select Devotion to read some reflections which are provided every day. When you first open the devotion, you must be connected to Internet, because the devotion materials are available online. However, when you connect, the devotion from that day to 30 days later will be downloaded and stored so you do not need to be connected to the Internet to read the devotion of the following days.

Press the Switch button to read devotions from different sources. Press the left arrow or right arrow buttons to move between days. Open the menu to copy the contents of the devotion or to share them with others.

Some devotions contain verse addresses that you can press to jump directly to the verses.

Songs

This feature is only available in the Indonesian version of Bible for Android (search for "Alkitab" in Android Market).

Press MENU and select Songs to read Christian songs.

Press Search to open the song list. To get the newest song data, erase the old data by opening the menu at this screen.

Version

You can read another version of the Bible (in different languages or different translations) by selecting the Version menu. At the beginning, there will be only one version available. Tap on Other versions... to add and manage Bible versions.

In the managing versions screen, there will be some pre-set versions that are ready to be downloaded. You will need to download the data file for that version beforehand. The data file will be stored in the SD Card. Each version occupies around 5 MB of space. After you download them once, you don't need to download them again.

If you have PDB files in PalmBible+ format, those files can be read by this application. This application does not directly read .pdb files, instead it will convert the .pdb files to its own format whose file name ends with .yes beforehand. This file is stored on the SD Card or USB Storage in /bible/yes folder with the file name generated depending on the PDB file opened. Long-press on one of the items on the version list, and select Details to check the location of the files and so on.

The checkboxes on the right of the versions indicates whether the versions are shown on the version selection dialog.

Verse Menu

Tap on one or more verses so that the verses are selected, and then click the arrow button that appears.

  • Copy verse to copy the selected verses to be pasted into other applications.
  • Add bookmark to add or edit a bookmark of the verse.
  • Write note to add a note to the verse.
  • Highlight to put a highlighter color into the selected verses.
  • Share to send the selected verses by email, bluetooth, Twitter, Facebook, etc. (Depending on the application installed on your phone.)

If your phone has NFC (Near Field Communication) capabilities, you can show the same Bible verse you're reading on your friend's phone, which need to have NFC capability as well.

Touch the back of your phone to the back of your friend's phone, and you'll see the outer space view on the display. Tap the screen to send the verse address to your friend. Your friend's phone will perform one of these:

  • Jump to the verse you send, if the Bible application is already open.
  • Open the Bible app and go to the verse that you send, if the app is not open yet.
  • Open the Bible app page on Google Play so that your friend can download and install the app.
